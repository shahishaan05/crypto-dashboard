# 💹 Crypto Dashboard

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![API](https://img.shields.io/badge/API-CoinGecko-green)

A modern **cryptocurrency dashboard web application** that displays **live crypto prices** using the CoinGecko API.

The app includes a **dashboard view** for popular cryptocurrencies and a **market page** showing multiple coins.

---

# 🚀 Features

* 💰 Live cryptocurrency prices
* 🔄 Refresh prices button
* 📊 Dashboard view for top coins
* 📈 Market page with multiple coins
* 🌐 API integration
* 🎨 Clean and modern UI

---

# 🌍 API Used

CoinGecko API

Example request:

```
https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum,dogecoin&vs_currencies=usd
```

# ▶️ Installation

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/crypto-dashboard.git
```

```
index.html
```

in your browser.
